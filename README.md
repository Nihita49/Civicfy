# Civicfy
Overview

Civicfy is a web-based platform designed to empower citizens to report civic issues in their locality quickly and efficiently. The system allows users to submit complaints such as garbage overflow, water scarcity, damaged roads, and electricity issues, helping authorities identify and resolve problems faster.

The platform promotes community participation, transparency, and accountability in urban governance.
Key Features
Issue Reporting
Users can submit civic complaints with:
Issue title
Media upload (images, audio, video)
Voice reporting option
Location tagging
Category classification
Additional description
AI-assisted Classification

Issues are categorized automatically into categories such as:
Garbage
Water
Road
Electricity
Media Evidence

Users can attach:
Images/Videos recordings: This helps authorities better understand the problem.
Voice Reporting: Users can report issues through voice input, making the platform accessible for people who may find typing difficult.
Issue Tracking: Each complaint generates a unique tracking ID so users can monitor the progress of their report.
Community Engagement:The platform includes community interaction features such as:Likes,Comments,Visibility of reported issues This encourages civic participation.
